# Task Template

## Goal
One sentence describing the desired outcome.

## Context
Relevant files, architecture notes, and current behavior.

## Constraints
- Languages/frameworks
- Performance/security constraints
- Forbidden changes

## Acceptance Criteria
- [ ] Behavioral outcome A
- [ ] Tests added/updated
- [ ] Lint/format checks pass
- [ ] Docs updated if needed

## Out of Scope
List explicitly what should not be changed.

## Validation Commands
Commands the agent must run before completion.

## Deliverables
- File changes expected
- PR summary expectations
